## Download

Grab **`MovX-Setup.exe`** from the [Releases](https://github.com/Kishanx08/movx/releases) page.

## Install

1. Run `MovX-Setup.exe`
2. If Windows SmartScreen shows a warning, click **More info → Run anyway**
3. Follow the installer prompts
4. Launch **MovX** from your Start Menu or desktop shortcut

## FAQ

**Why does Windows warn me before installing?**
The app isn't distributed through the Microsoft Store, so Windows shows a standard first-run warning for that. It's safe to proceed.

**Where do I report a bug?**
Open an [issue](https://github.com/Kishanx08/movx/issues) with your Windows version and what happened.
